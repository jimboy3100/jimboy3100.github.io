About Mod

Author: Jimboy3100 (Developer on java, vb, c++, asp.net, javascript and sql libraries)
User Account Jimboy3100 for gmail, hotmail, skype, facebook, twitter, discord, youtube, github, joomla, etc

This mod combines many scripts for agario game. It is based on ogario.ovh platform with all features,
and uses script codes such as auto-respawn, kitty insist reconnects for ffa/exp tokens and plenty
of other scripts that can be found on greasefork website. It works only as javascript / jquery
design palette script that works on the users terminal, using dynamic libraries from cloud,
but not sending any user information to the web. It also does not use any cheat codes,
but provides mathematical / programming scripts that help users play with comfort.
It also provides better ogario.ovh syncing support with the game and some basic game bug fixes to the game itself
It also:
Auto Updates
Provides many features shown on the feature section
Is encoded
Has the best scripts and features than any other mod for agar.io (as far as i know)
Provides vanilla-like framework with one button click, for users who are used to vanilla game

LME Chat Commands

1. “Send a new agario server” - send your agario server link to users and suggest them to join
Message format: [srv]agario URL[/srv]
Example 1: To send this server: http://agar.io/#2DUZJJ
put in the message: [srv]http://agar.io/#2DUZJJ[/srv] 


Example 2: To send this server: http://agar.io/?sip=live-arena-1dzn7sm.agar.io&?r=EU-London&?m=
put in the message: [srv]http://agar.io/?sip=live-arena-1dzn7sm.agar.io&?r=EU-London&?m=[/srv] 


and to embed password "mypassword": [srv]http://agar.io/?sip=live-arena-1rdujzy.agar.io&?pass=mypassword&?r=EU-London&m=[/srv]


*Due to limit of 60 chars on chat texts, users may experience less chars on the box, command still works


2. “Show a URL on chat” - send your url link that users can visit on a new tab.
Message format: [url]URL[/url]
Example: To send this hyperlink: http://google.com
put in the message: [url]http://google.com[/url]


3. “Suggest a video on chat” - send a youtube video on chat and suggest users to use it on mod
Message format: [yut]URL[/yut]
To suggest this video: https://www.youtube.com/watch?v=dQw4w9WgXcQ
put in the message: [yut]https://www.youtube.com/watch?v=dQw4w9WgXcQ[/yut]


4. “Show images on chat” - enables images from imgur.com or hizliresim.com
Message format: [img]URL[/img]
To send this image: http://i.imgur.com/mO44ZE3.jpg
put in the message: [img]http://i.imgur.com/mO44ZE3.jpg[/img]


5. “Show videos on chat” - enables youtube videos.
Message format: [yt]ID[/yt]
To send this video: https://www.youtube.com/watch?v=dQw4w9WgXcQ
put in the message: [yt]dQw4w9WgXcQ[/yt]


6. “Suggest a Skype clan / group on chat” - suggest users to join the Skype group that you are in
Message format: [skype]URL:Skype Protocol [/skype]
To suggest this skype group: https://join.skype.com/qcucP8fXvOP8
put in the message: [skype]https://join.skype.com/qcucP8fXvOP8[/skype]


7. “Suggest a Discord clan / group on chat” - suggest users to join the Discord server that you are in
Message format: [discord]URL:Discord Protocol [/discord]
To suggest this Discord server: https://discordapp.com/invite/wZuSDx
put in the message: [discord]https://discordapp.com/invite/wZuSDx[/discord] or [discord]https://discord.gg/wZuSDx[/discord]

Virtual Bots

Steps:
1. Be sure you use the latest Version of LM Express or LM Tiny OpenSource
2. Install I'm not robot captcha clicker Extension for chrome
3. Logout (Login on your normal window after you load bots)
4. Copy the url address you play on a new link e.g http://agar.io/?sip=live-arena-1mpbb6h.agar.io
5. Add this command at the end ?mod=tiny&?bots=YES e.g: http://agar.io/?sip=live-arena-1mpbb6h.agar.io?mod=tiny&?bots=YES
For Tiny mod to be enabled without autoplay add this command at the end ?mod=tiny e,g http://agar.io/?sip=live-arena-1mpbb6h.agar.io?mod=tiny
Look at Auto respawn and Randomized movement Settings
6. If not automatically clicked play, click "PLAY" for the first game
7. Open more pages - more Bots
If reCaptcha start using icons, there is not much you can do. You may probably find an Extension which clicks icons for reCaptcha
Bots are illegal, use at your own risk

Improving Performance

Legend Mod has Unlimited FPS unlocked (quicker than Vanilla)

The Preliminary:

Legend Express does not load skins and profile's libraries, thus the performace is significantly greater than agar.io
Ethernet cable internet has the most important role on connection Ping, prefer Ethernet than WIFI
Call to provider and refer to enable “fast pack”, mention that you play internet game, it is a free service. Gamers need quick response (PING)
Update Chrome to latest version: chrome://settings/help
Update Legend Mod to latest version from www.legendmod.ml
Go to chrome://flags/ , ensure that Override software rendering list is "Enabled" (you'll need to relaunch Chrome for any changes to take effect)
Go to chrome://settings/system , ensure the Use hardware acceleration when available checkbox is checked (you'll need to relaunch Chrome for any changes to take effect)
Go to chrome://flags, Ensure that Disable WebGL is not activated (you'll need to relaunch Chrome for any changes to take effect). This feature might not be shown, as already been enabled on new Chrome Updates

Conflicts:

Some players experience conflicts that cause great FPS drop, bugs or even crash.Top reasons:
Ad blockers, (Must be disabled)
Sometimes Chrome enables its own Ad blocker that disables cookies. You must even disable all blocks to other cookies (youtube, miniclip, xsolla, etc) because they affect agario
Cookie blockers, (Must be disabled)
Chrome Extensions, chrome://extensions/ , Other extensions rather than Tampermonkey, AgarioScripts must be disabled. Not both Tampermonkey and AgarioScripts Userscript managers used
Other Tampermonkey (top right button->dashboard) or AgarioScript Userscripts. Use 1 Userscript at a time

Performance notes:

Top performance killers which should be OFF on a low-end PC:
Ghost Cells
Skins
Jelly Physics
Skins on teamboard
Background sectors
Background grid
Rainbow food
Names (disable names or use “Auto hide name” option)

Some tips to get more stable FPS:

Prefer joining locked Servers by typing tokens directly to address bar e.g "http://agar.io/?sip=151.80.98.61:1526&?r=EU-London&?m="
Use “Optimized names” and “Optimized mass” option
Use “Auto hide mass” option
Use “Short mass” option
Disable auto zoom
Choose agar.io servers that are close to you
Do not use any kind of helpers if you don’t need them. Less to draw means better performance
Try different browsers (Chrome, Opera, Vivaldi)

Also prefer using:

talky.io/ voice chat tools, from the game itself (“Voice and Camera Chat” button).
It makes chat rooms per IP/token, per password tag. This way we get rid of the extra information that discord/raidcall/teamspeak/skype voice rooms send
Users of other mods or Vanilla, can use the utility and be compatible, if they use this script:
https://jimboy3100.github.io/Voicechat.user.js or Agario Voice Chat rooms - GreaseFork
