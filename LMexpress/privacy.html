Privacy Policy for Legend Express

Last updated: February 25, 2026

1. Introduction

Legend Express is a Chrome extension designed for the game Agar.io. We are committed to protecting your privacy.

2. Data Collection

Legend Express does not collect, store, or transmit any personal user data to external servers. All extension settings and game preferences are stored locally in your browser using the Chrome Storage API.

3. Permissions

  • Storage: Used solely to save your mod configurations and hotkeys locally.
  • Host Permissions: Used to fetch mod components and assets from jimboy3100.github.io to enable the modded gameplay experience.

4. Third-Party Scripts

The extension may load modded scripts (Legend Mod, Ogario) from the official jimboy3100.github.io repository. These scripts are community-maintained and used to modify the agar.io gameplay interface.

5. Changes to the Policy

We may update this policy if technical changes require it. Updates will be posted on this page.

6. Contact

If you have questions, contact the developer via GitHub: https://github.com/jimboy3100
